Хромоматематическое моделирование 2D объектов.

После выставления опций модели, необходимо нажать на кнопку "Расчет", для вывода результата.

Опции моделирования:
1. Цветовая модель
  1.1 HMM_N - черно-белое окрашивание чисел
  1.2 HMM_DN - цветное градиентное окрашивание
2. Density (плнотность) - количество отображаемых точек на графике
3. xMin - минимальное значение на оси X
4. xMax - максимальное значение на оси X
5. yMin - минимальное значение на оси Y
6. yMax - максимальное значение на оси Y
7. Axes чек-бокс - показывать оси координат (X и Y)
8. Коэффициента А - нормирующий коэффициент для формулы Z = Ker(A *SIN(x + y)), где Ker - ядро числа a, т. е. рекурсивная сумма цифр.
