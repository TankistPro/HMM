Хромоматематическое моделирование на последовательности четных числе.

После выставления необходимых опций модели, необходимо нажать на кнопку "Расчет", для вывода результата.

Опции моделирования:
1. Цветовая модель
  1.1 HMM_N - черно-белое окрашивание чисел
  1.2 HMM_DN - цветное градиентное окрашивание
2. Тип визуализации:
  2.1 Последовательная - числа располагаются друг за другом, начинаю с левого верхнего угла.
  2.2 Спираль - числа располагаются начиная с центра, образуя спираль с поворотом направо (против часов стрелки)
3. Метод окрашивания:
  3.1 Линейный - квадратики последовательно окрашиваются в цвета, согласно выбранной цветовой модели
  3.2 Хеширование - окрашивание происходит с помощью хеш-функции
  3.3 Золотое сечение - вычисление цвета происходит, с помощью алгоритма вычисления золотого сечения
4. Aspect - масштаб изображения
5. Кол-во цветов - необходимое количество цветов, для окрашивания текущей модели
6. Показывать значения - чек-бокс, позволяющий отображать у каждого квадрата его числовое значение
